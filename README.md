# ⚡ Electrics - Electrical Services Web App
A Flask-based web application for managing electrical services, bookings, and technicians.

## Features
- Customer & Technician dashboards
- Booking management
- AI-powered chatbot (Google Gemini)
- Admin panel

## Tech Stack
- Python / Flask
- SQLAlchemy (SQLite)
- Google Gemini AI
- HTML / CSS / Jinja2
